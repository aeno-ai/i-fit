# i Fit

**Course / Project**: Gym exercise guide — UP-FBS2-BSIT1-02

**Programmer**: Rabaya, Ron Arjie E.

---

## What it is

A clean, responsive fitness website that teaches exercises for different muscle groups, provides basic nutrition guidance, and includes simple a interactive tool BMI calculator. Designed to be user-friendly and accessible.

---

## Core features

* **Exercise Library** — categorized by muscle group and fitness goal, with images or short videos showing proper form.
* **Interactive Tool** — BMI calculator implemented in JavaScript.
* **Nutrition Guide** — meal suggestions, pre/post-workout tips, and optional supplement notes.
* **Responsive UI** — works across common screen sizes; lightweight animations and hover effects.
* **Accessibility & HCI** — readable layouts, clear navigation, optional dark mode, and non-intrusive animations.

---

## HCI considerations (how they apply)

* **User-centered design**: simple navigation, prominent calls-to-action, and consistent layouts.
* **Readability**: large headings, legible contrast, and concise copy for quick scanning.
* **Feedback & affordances**: clear button states, previews for exercises, and confirmation before starting a plan.
* **Accessibility**: semantic HTML, alt text for images, keyboard navigation, and an optional dark theme.
